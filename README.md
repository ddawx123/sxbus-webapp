# Smart Public Traffic Of Shaoxing
## 绍兴智慧公共交通云平台

本项目设计模式为webapp，待主体功能完善后将尝试发布nativeapp。

本项目目前支持以下特性：
- 1、支持公交站台所有线路的查询
- 2、支持查看指定公交线路所有车辆的实时GPS定位信息

目前该项目尚未解决跨区公交的查询，所以暂时只能查询市公交集团（越城区）所隶属的所有线路。柯桥区、上虞区等地线路查询将在后续更新中逐步开放！
